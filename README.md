# 📰 Fake News Detection System

A Machine Learning project that detects whether a news article is **Fake or Real** using NLP and 4 ML models.

## 🔍 Models Used
- Logistic Regression
- Decision Tree
- Gradient Boosting Classifier
- Random Forest

## 🛠 Tech Stack
Python | Scikit-learn | Pandas | TF-IDF | Gradio | Jupyter Notebook

## 📁 Dataset
- `Fake.csv` — Fake news articles (label: 0)
- `True.csv` — Real news articles (label: 1)
> Dataset source: [Kaggle Fake News Dataset](https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset)
